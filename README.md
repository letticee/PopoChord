# PopoChord
## Introduction
Popochord 是一款作為和弦演進標記的應用程式。當靈感乍現，使用者可以透過 PopoChord 錄下旋律，藉由時間點的標記，在 chord progression 頁面，進行和絃進行的排列。並且，我們提供一個依據調性和聲的篩選工具，讓使用者清楚的了解自己所在的調性，對於和絃色彩的配置，與和弦進行的合理性，提供更直觀的顯示。
